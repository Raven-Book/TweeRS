# 宏

import { TweePlayground } from '@/components';

宏（Macro）是 SugarCube 的核心功能。用 `<<宏名>>` 的语法调用，可以实现逻辑控制、变量操作、显示内容等。

## 什么是宏

前面章节中的链接语法 `[[]]` 只能做简单跳转。宏则能做更多事情——设置变量、条件判断、循环等。

宏的基本格式：

```twee
<<宏名 参数>>
```

有些宏是成对使用的，用 `<</宏名>>` 结束：

```twee
<<宏名>>
  内容
<</宏名>>
```

## 常用宏一览

后续章节会逐一讲解这些宏：

| 宏 | 用途 | 需要闭合 |
|------|------|:------:|
| `<<set>>` | 设置变量 | 否 |
| `<<print>>` | 输出表达式结果 | 否 |
| `<<if>>` | 条件分支 | 是 |
| `<<for>>` | 循环 | 是 |
| `<<link>>` | 创建可执行代码的链接 | 是 |
| `<<button>>` | 创建按钮 | 是 |
