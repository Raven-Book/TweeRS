# 方糖 (SugarCube)

import { TweePlayground } from '@/components';

SugarCube 是最流行的 Twine 故事格式之一。它提供了丰富的宏（macro）系统、变量管理和内置 UI，适合制作从简单到复杂的文字游戏。

## 它能做什么

- **文本排版** — 粗体、斜体、列表、标题等
- **链接导航** — 让玩家在片段之间跳转
- **变量系统** — 用 `$变量` 记录玩家状态（血量、金币、选择……）
- **条件分支** — 根据变量值显示不同内容
- **循环** — 批量生成重复内容

## 最小示例

下面是一个完整的 SugarCube 故事，只有一个片段：

<TweePlayground code={`:: StoryData
{
  "ifid": "D674C58C-DEFA-4F70-B7A2-27742230C0FC",
  "format": "SugarCube",
  "format-version": "2.37.3"
}

:: StoryTitle
方糖示例

:: Start
你好，世界！
`} />

`:: Start` 是故事的入口片段。SugarCube 会自动从这里开始。
