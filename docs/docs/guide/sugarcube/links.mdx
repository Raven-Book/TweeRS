# 链接与导航

import { TweePlayground } from '@/components';

链接是 Twine 故事的核心——玩家通过点击链接在片段之间跳转。

## 基础链接

用 `[[片段名]]` 创建链接，点击后跳转到对应片段：

<TweePlayground hideStoryHeader code={`:: StoryData
{
  "ifid": "D674C58C-DEFA-4F70-B7A2-27742230C0FC",
  "format": "SugarCube",
  "format-version": "2.37.3"
}

:: StoryTitle
链接示例

:: Start
你站在路口。

[[向左走]]
[[向右走]]

:: 向左走
你来到了一片花园。

[[回到路口|Start]]

:: 向右走
你来到了一条小河边。

[[回到路口|Start]]
`} />

## 显示文本与目标分离

`[[显示文本|目标片段]]` 可以让链接显示的文字和跳转目标不同。

上面示例中的 `[[回到路口|Start]]` 就是这种用法——显示「回到路口」，但跳转到 `Start` 片段。

也可以用箭头语法：`[[显示文本->目标片段]]`，效果相同。

## 练习
