# TweeRS

用文本文件构建 Twine 互动小说游戏。

```bash
tweers build story/ -o game.html
```

## 为什么用 TweeRS？

Twine 官方编辑器是图形界面，适合小型项目。但当项目变大时，你可能需要：

- 用 Git 管理故事版本
- 用熟悉的代码编辑器写作
- 自动化构建和发布流程

TweeRS 让你用纯文本 `.twee` 文件写故事，然后编译成可运行的 HTML 游戏。
